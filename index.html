Welcome to PACER
How much do you want to see? You can switch anytime from the header.
Glossary
Plain-language definitions for every term in PACER.
Cloud vs Codex
Cloud is Anthropic / Claude Code (this app's main subject). Codex is OpenAI's coding tool you run through ChatGPT. The switch at the top flips the dashboard between each service's live limits and routing.
5-hour limit
A rolling cap that resets every 5 hours. Hit it and you're paused until it refreshes, even when your weekly budget still has room.
Weekly limit
Your larger budget that resets once a week (Friday 06:00 Guatemala time for Claude). The main number to pace against.
Burn rate
How fast you're spending budget, shown as % per hour. Compare it to the required pace to know whether you're ahead or behind.
Pacing plan
A saved target for the week: how much budget to spend per day so you don't run out early.
Hidden cost audit
A scan for silent token drains — oversized context, repeated tool calls, cache misses — that cost budget without obvious work.
Run full audit
Triggers that scan on demand and reports what's quietly eating your budget.
Circuit breaker
A guardrail: when your burn would blow the budget before reset, it warns you (and can suggest pausing) so you don't hit a hard stop mid-task.
Reset alert
A countdown and heads-up before a 5-hour or weekly reset, so you can time heavy work for right after a refresh.
Pattern detection
Looks across your history for recurring habits — heavy Mondays, late-night spikes — so you can adjust.
Yield analysis
Measures value per token: which sessions were worth the spend. Import a CodeBurn or Claude session export to get a verdict.
Pacing simulator
A what-if calculator. Set current usage %, hours until reset, work hours per day, and sessions remaining; it projects when you'll hit the limit.
Profile compare
Pre-built usage archetypes (light pro, heavy pro, max balance, heavy max, code power) to benchmark your habits against.
Broker & intents
A router that splits your weekly budget across intents (income, build, explore) and assigns each to Cloud or Codex, protecting priority work when budget tightens.
Plan tier (Pro / Max / API)
Which Anthropic subscription you're on. It sets the limit math PACER paces against; switch it in the header to match your plan.
Fallback Usage Update
Fallback only. Use this when Tokemon is offline; PACER computes burn and reset time from live history when available.
Log Task Routing Decision
Track which tool handled a task. Stored locally only.
